# 🚦 Smart Traffic AI System

An AI-based smart traffic management system designed to optimize traffic flow and signal timings dynamically using computer vision.

## 🚀 Features
- **Real-time Vehicle Detection:** Accurately counts vehicles using AI.
- **Dynamic Signal Control:** Optimizes traffic light timings based on vehicle density.
- **Streamlit Dashboard:** Interactive and user-friendly web interface.
